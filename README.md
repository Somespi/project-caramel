# What is Caramel?

The philosopher John Locke popularized the idea of tabula rasa, suggesting that people are born without any knowledge and gain understanding through their experiences.

Furthermore, Charles Darwin argued that mental abilities evolved gradually rather than appearing suddenly, suggesting continuity between humans and other animals.

Following this concept, Caramel starts without any pre-existing knowledge of its surroundings, but a small set of basic reflexes often seen in newborn animals. These reflexes serve as essential to interact with the world, not as learned knowledge. This setup allows the robot to develop more complex behaviors over time through experience.  



# Why?

Caramel explores this research question:

> **To what extent can a behaviorist robot learn to imitate observed behaviors and generate new actions guided by minimal innate reflexes and intrinsic motivational systems?**

By investigating this question, the project aims to understand how far robots can go with experience-based learning and which behaviors need an inherent structure versus those that can develop through interacting with their environment through curiosity, and "hormones."
